## Install  
[安装依赖]  

```bash
node >= 4.0.0
```

```bash
$ npm install -g dva-cli yarn
$ yarn install
```

## Usage  
[用法]  

#### development[开发环境]  

```bash
$ yarn start
```
访问：[http://127.0.0.1:8000](http://127.0.0.1:8000)  

#### production[生产环境]

```bash
$ yarn run build
```
